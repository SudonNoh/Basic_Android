# 리소스(Resource)
    - drawable
        - 이미지
            - 실제 이미지 파일, xml
        - 동일한 이미지를 해상도 별로 여러 장을 준비할 수 있다
            - 핸드폰의 해상도에 맞는 이미지를 사용할 수 있다.
            - 안드로이드 os가 핸드폰 해상도에 적합한 이미지를 알아서 선택
    - layout
        - 화면을 구성하는 xml
    - mipmap
        - 앱 아이콘
    - values
        - colors
            - 자주 사용되는 색에 이름을 붙여 관리
            - RGB(Red, Green, Blue)
            - 16진수
            - 10A 11B 12C 13D 14E 15F
        - string
            - 문자열
